== Description ==

The Portfolio tab displays all of your positions for the contracts on the current active window.


== Installation ==

Installation is very simple 

e.g.

1. Upload `portfolio-with-tab to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats it, The Portfolio tab displays all of your positions for the contracts on the current active window.
